# Product-catalog
Тестовый проект на React. 

## Описание
Приложение демонстрирует:
- адаптивное мобильное меню
- категории товаров
- кнопки с активным состоянием
- возможность добавлять звуки при кликах

## Установка
1. Склонируйте репозиторий:
   
git clone https://github.com/Aleksei-Sabynich/Product-catalog.git
cd Product-catalog

2.Установите зависимости:
npm install


## Запуск
Запустите проект:

Локально: npm run dev
Для продакшн-сборки: npm run build
